# How to Add Your Documents

Drop your PDF files into this folder using these exact filenames:

| File | Purpose |
|------|---------|
| `personal-statement.pdf` | Personal statement |
| `resume.pdf` | Resume |
| `cv.pdf` | Curriculum vitae |
| `rec-letter-1.pdf` | First recommendation letter |
| `rec-letter-2.pdf` | Second recommendation letter |
| `rec-letter-3.pdf` | Third recommendation letter |

After dropping a file here, it will immediately appear in the browser preview
on the Professional page — no code changes needed.

## To add more recommendation letters

1. Add the PDF here as `rec-letter-4.pdf`, etc.
2. Open `professional.html` and copy one of the `.rec-item` blocks
3. Update the `data-file` attribute, name, and role text

## To update a document

Simply replace the file with a new one using the same filename.
